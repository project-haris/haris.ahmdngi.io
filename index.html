Core Capabilities

Built for Maritime Security

A comprehensive edge computing solution designed for the unique challenges of maritime environments.

📡

NMEA Data Collection

Native support for NMEA 2000 via CAN bus and NMEA 0183 via serial. PiCAN-M HAT provides marine-grade connectivity to vessel networks.

🔒

Encrypted Transmission

Tailscale ensures end-to-end encryption via WireGuard. All data transmitted to the MSOC is secured with modern cryptographic protocols.

⚡

Edge Processing

Raspberry Pi 5 with NVMe storage enables local processing and caching. Signal K server normalizes data before transmission.

📊

ELK Stack Integration

Elasticsearch, Logstash, and Kibana provide local visualization and log management. Filebeat ships logs to central servers.

🛡️

Tamper Evidence

Physical security seals and monitoring detect unauthorized access. Automatic data wipe on compromise detection.

🤖

Automated Deployment

Ansible role for reproducible deployments. Transform a vanilla Raspberry Pi OS into a fully functional security node.

Quick Start

Deploy in Minutes

Automated deployment via Ansible. From vanilla OS to operational security node.

1

Prepare Hardware

Flash Raspberry Pi OS to NVMe, install PiCAN-M HAT, connect to vessel network.

2

Configure Variables

Set vessel ID, credentials, and network settings in Ansible host_vars.

3

Run Playbook

Execute Ansible playbook to deploy the complete stack automatically.

4

Verify & Seal

Test data flow, access dashboards, seal enclosure, deploy to vessel.
